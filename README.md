# 🎬 Movie Recommender System

A content-based movie recommender system built with **Streamlit**.
Select any movie and get **5 similar movie suggestions** instantly!

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📁 Dataset
Download from [Kaggle TMDB Dataset](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
